Among other things, this project requires you to demonstrate a basic knowledge of frequency distributions and histograms. In this project you will write Python code to create the data for a frequency distribution chart that emulates a histogram for a given dataset. You will write that code without importing any libraries.

PROGRAM SPECIFICATIONS

Write and submit a Python module named Proj02Runner.py that works in conjunction with the driver source code contained in the file named Proj02.py to produce the output illustrated by Figure 1, Figure 2, and Figure 3 below on the command-line screen.

Be sure that your name appears in the certification as indicated.

The code that you write must not import anything.

The program must terminate and return control to the operating system when control reaches the end of the file named Proj02.py. In other words, infinite loops are not allowed.

Do not copy the code from the file named Proj02.py into your file named Proj02Runner.py.

You must not modify the code in the file named Proj02.py. Your instructor will execute the code in his original version of the file named Proj02.py in the same folder as your file named Proj02Runner.py for the purpose of grading your assignment. If you modify the code in your copy of the file named Proj02.py, it is unlikely that your program will run successfully when tested.

The required output depends on three command-line arguments that are entered when the program is run. The command-line arguments are used to create a dataset. Three sample outputs with different command-line arguments are shown below. You can use those sample outputs to test your program.

Your instructor will use a different set of command-line arguments than any of the three shown below to create a dataset and grade your program. Your code must be able accommodate a different dataset regardless of the number and/or range of values in the dataset.

You should begin by analyzing the code in the file named Proj02.py. In particular, analyze the construction of the dataset in the list named data to get an idea of what you will be working with. That list, which is shown at the top of each sample below, will be passed to your function named run in a class named Runner in a file named Proj02Runner.py. Before writing any code, you should try to replicate the charts shown below with pencil and paper using the dataset shown at the top of each sample. Once you can do that, you should be able to convert that process into code.

Your challenge is to process that incoming dataset in such a way as to return a list which, when processed by the code in the file named Proj02.py, will create a frequency distribution chart that emulates a histogram.
