PROGRAM SPECIFICATIONS

Write and submit a Python module named Proj03Runner.py that works in conjunction with the driver source code contained in the file named Proj03.py to produce the output illustrated by Figure 1 and Figure 2 below.

Be sure that your name appears in the certification as indicated and also at the top of the output image as indicated.

Do not copy the code from the file named Proj03.py into your file named Proj03Runner.py.

Your instructor will execute the code in his original version of the file named Proj03.py in the same folder as your file named Proj03Runner.py for the purpose of grading your assignment. If you modify the code in your file named Proj03.py, it is unlikely that your program will run successfully when tested.

Specifications for the output image

The output image must be a single image containing an independent display in each of four quadrants as shown in Figure 1.

Scatter plot

The upper-left quadrant must display a noisy relationship between two random variables exhibiting a positive correlation similar to that shown in Figure 1. Create and plot approximately 100 data pairs and draw the best fit line as shown.

Standard normal probability density function

Plot the standard normal probability density function from minus three sigma to plus three sigma in the upper-right quadrant as shown in Figure 1.This must be an exact copy of (not just similar to) that shown in Figure 1.

Histogram for positively skewed data

The lower-left quadrant must display a relative frequency histogram for data with a positively skewed distribution similar to that shown in Figure 1. Use a random number generator to create the data being plotted in the histogram. Cause the data values to range from zero to 2000 and cause your histogram to have 50 bins. Generate enough data to ensure that the histogram is relatively smooth with no gaps indicating empty bins.

Cumulative distribution function of the standard normal probability density function

Plot the cumulative distribution function of the standard normal probability density function from minus three sigma to plus three sigma in the lower-right quadrant as shown in Figure 1. This must be an exact copy of (not just similar to) that shown in Figure 1.
